# 📁 Sistema de Archivo Fiscal

## 📌 Descripción
Sistema web desarrollado en PHP y MySQL para la gestión de carpetas fiscales, permitiendo su registro, ubicación, préstamo y control de vencimientos.

## 🚀 Funcionalidades
- Registro de carpetas fiscales
- Búsqueda por número de carpeta
- Préstamo de carpetas
- Generación de guía de préstamo
- Control de vencimientos
- Alertas de retraso
- Nota de devolución
- Importación masiva desde Excel con validaciones

## 🛠️ Tecnologías
- PHP
- MySQL
- HTML / CSS
- XAMPP
- PhpSpreadsheet

## 📊 Base de Datos
Tablas principales:
- carpeta_fiscal
- prestamo
- detalle_prestamo
- dependencia
- devolucion
